# 解题过程

## 第（I）问

### Step 1
- 标题：由直角三角形求 D 点坐标
- 目标：求出 OD 的长度，从而得到 D 的坐标。
- 推导：
  - ∵ C＝（0，2√3）
  - ∴ OC＝2√3
  - ∵ D 在 x 轴上，∠ODC＝60°
  - ∵ 在 Rt△ODC 中，tan60°＝OC/OD
  - ∴ √3＝2√3/OD
  - ∴ OD＝2
  - ∴ D＝（2，0）
- 当前结论：D＝（2，0）

### Step 2
- 标题：由垂直和等长求 E 点坐标
- 目标：利用 DE⊥CD、DE＝CD 求 E 的坐标。
- 推导：
  - ∵ OD＝2，OC＝2√3
  - ∴ CD＝√(2²＋(2√3)²)＝4
  - ∵ DE＝CD
  - ∴ DE＝4
  - ∵ CD 与 x 轴夹角为 120°，DE⊥CD
  - ∴ DE 与 x 轴夹角为 30°
  - ∴ 从 D 到 E，横向增加 4cos30°＝2√3，纵向增加 4sin30°＝2
  - ∴ E＝（2＋2√3，2）
- 当前结论：E＝（2＋2√3，2）

## 第（II）①问

### Step 1
- 标题：求 O'、C'、D' 坐标
- 目标：把平移后三角形的三个顶点用 t 表示。
- 推导：
  - ∵ △COD 沿 x 轴向右平移，OO'＝t
  - ∴ O'＝（t，0）
  - ∴ C'＝（t，2√3）
  - ∴ D'＝（t＋2，0）
- 当前结论：O'＝（t，0），C'＝（t，2√3），D'＝（t＋2，0）

### Step 2
- 标题：由边界位置判断四边形出现和结束的状态
- 目标：判断四边形 C'MDN 什么时候开始出现，什么时候结束。
- 推导：
  - ∵ 当 C' 与 C 重合时，C'D' 与 CD 重合，重叠部分还不是四边形 C'MDN
  - ∴ 左边界：C' 与 C 重合
  - ∵ C' 从 C 向右移动后，O'C' 与 CD 交于 M，C'D' 与 DE 交于 N
  - ∴ 此时重叠部分变成四边形 C'MDN
  - ∵ 当 O' 与 D 重合时，重叠部分变成三角形
  - ∴ 右边界：O' 与 D 重合
  - ∴ 0<t<2
- 当前结论：0<t<2

### Step 3
- 标题：由直角三角形求 S△O'DM、S△DND'
- 目标：求出从大三角形中减去的两个小三角形面积。
- 推导：
  - ∵ O'D＝OD－OO'＝2－t
  - ∵ ∠O'DM＝60°
  - ∴ O'M＝√3（2－t）
  - ∴ S△O'DM＝1/2×（2－t）×√3（2－t）＝√3（2－t）²/2
  - ∵ DD'＝t，△DND' 是直角三角形，∠DD'N＝30°
  - ∴ DN＝t/2，D'N＝√3t/2
  - ∴ S△DND'＝1/2×t/2×√3t/2＝√3t²/8
- 当前结论：S△O'DM＝√3（2－t）²/2，S△DND'＝√3t²/8

### Step 4
- 标题：由三角形面积相减求 S
- 目标：用大三角形减去两个小三角形，求出重叠面积。
- 推导：
  - ∵ S＝S△O'D'C'－S△O'DM－S△DND'
  - ∵ S△O'D'C'＝1/2×2×2√3＝2√3
  - ∵ S△O'DM＝√3（2－t）²/2，S△DND'＝√3t²/8
  - ∴ S＝2√3－√3（2－t）²/2－√3t²/8
  - ∴ S＝2√3t－5√3t²/8
  - ∴ 0<t<2
- 当前结论：S＝2√3t－5√3t²/8，0<t<2

## 第（II）②问

### Step 1
- 标题：由图形变化判断面积分段
- 目标：确定 1≤t≤3 中需要讨论的图形形态。
- 推导：
  - ∵ 当 1≤t<2 时，重叠部分仍按四边形 C'MDN 的面积公式计算
  - ∵ 当 t＝2 时，原点 D 与 O' 重合，是两种形态的分界
  - ∵ 当 2≤t≤3 时，重叠部分变为三角形
  - ∴ 面积 S 需要分成 1≤t<2 和 2≤t≤3 两段讨论
- 当前结论：分段为 1≤t<2 和 2≤t≤3

### Step 2
- 标题：在 1≤t<2 中求四边形面积的极值
- 目标：用第（II）①问公式找出这一段四边形面积的最大值和最小值。
- 推导：
  - ∵ S＝2√3t－5√3t²/8
  - ∴ S＝√3（-5/8 t²＋2t）
  - ∵ 二次函数开口向下，对称轴为 t＝8/5
  - ∵ 1≤8/5<2
  - ∴ 当 t＝8/5 时，S 取得这一段最大值
  - ∴ Smax＝√3（-5/8×64/25＋16/5）＝8√3/5
  - ∵ t＝1 时，S＝11√3/8；t 趋近 2 时，S 趋近 3√3/2
  - ∴ 这一段最小值在 t＝1 时取得，Smin＝11√3/8
- 当前结论：这一段最大值为 8√3/5，最小值为 11√3/8

### Step 3
- 标题：在 2≤t≤3 中求三角形面积的极值
- 目标：求右侧三角形形态下的面积表达式，并找出这一段的最大值和最小值。
- 推导：
  - ∵ 2≤t≤3 时，重叠部分是三角形 C'KN
  - ∵ K 在 DE 上，O'K 与 x 轴垂直
  - ∵ 在 Rt△O'DK 中，∠O'DK＝30°，O'D＝t－2
  - ∴ O'K＝(t－2)/√3
  - ∴ C'K＝2√3－(t－2)/√3＝(8－t)/√3
  - ∵ △C'KN 是直角三角形，且 ∠C'NK＝60°
  - ∴ C'N＝√3C'K/2＝(8－t)/2
  - ∴ KN＝C'K/2＝(8－t)/(2√3)
  - ∴ S△C'KN＝1/2×(8－t)/2×(8－t)/(2√3)＝√3(8－t)²/24
  - ∵ 在 2≤t≤3 中，S 随 t 增大而减小
  - ∴ 当 t＝2 时，S 取得这一段最大值，Smax＝3√3/2
  - ∴ 当 t＝3 时，S 取得这一段最小值
  - ∴ Smin＝√3×25/24＝25√3/24
- 当前结论：这一段最大值为 3√3/2，最小值为 25√3/24

### Step 4
- 标题：比较不同范围极值求 S 范围
- 目标：比较两段图形对应的极值，写出 S 的范围。
- 推导：
  - ∵ 1≤t<2 时，最大值为 8√3/5，最小值为 11√3/8
  - ∵ 2≤t≤3 时，最大值为 3√3/2，最小值为 25√3/24
  - ∵ 8√3/5＞3√3/2，11√3/8＞25√3/24
  - ∴ 25√3/24≤S≤8√3/5
- 当前结论：25√3/24≤S≤8√3/5
