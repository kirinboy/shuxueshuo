# 可视化步骤设计

整体使用坐标系展示抛物线、坐标轴交点、旋转得到的 \(D\)、中点 \(E\)、竖直线上的 \(F\)，以及第②问中的平行四边形 \(MFDB\) 和点 \(P\)。

# 全局图层

- 坐标网格始终显示。
- 第（I）问显示固定抛物线 \(y=-x^2+2x+3\)，并标出 \(A(-1,0)\)、\(B(3,0)\)、\(C(0,3)\)。
- 第（II）问使用参数 \(m\) 控制图形，默认展示在满足平行四边形条件的族：
  \[
  y=-\frac3{2m}x^2+\frac32x+3m.
  \]
- 全局可视范围取 `domain = { minX:-1.4, maxX:7.0, minY:-1.0, maxY:10.5 }`，既能容纳第①问的 \(F(3,9)\)，也能容纳第②问 \(m=\frac34\) 的局部图形。

# 步骤规划

## i1s1：第（I）问，分解解析式求 \(A,B\)

- 显示固定抛物线、\(x\) 轴交点 \(A,B\) 和 \(y\) 轴交点 \(C\)。
- 用坐标标签突出 \(A(-1,0)\)、\(B(3,0)\)、\(C(0,3)\)。

## q1s1：第（II）①，根与系数求 \(a\)

- 设 \(A(-m,0)\)、\(B(n,0)\)，显示 \(C(0,3m)\)。
- 用表达式说明 \(y=a(x+m)(x-n)\)，并由 \(C\) 得 \(a=-3/n\)。
- 锁定 \(m=3\)，图上标出 \(A(-3,0)\)、\(B(6,0)\)、\(C(0,9)\)。

## q1s2：第（II）①，旋转与中点确定 \(D,E,F\)

- 显示线段 \(AC\)、旋转后的 \(CD\)、线段 \(AD\)。
- 标出 \(D(9,6)\)、\(E(3,3)\)、\(F(3,9)\)。
- 用竖直线段 \(EF\) 突出 \(EF=6\)。

## q2s1：第（II）②，平行四边形推出 \(n=2m\)

- 显示 \(M,F,D,B\) 四点和四边形 \(MFDB\)。
- 锁定 \(m=\frac34\) 附近状态，以便图形紧凑。
- 显示对边平行关系：\(FD\parallel MB\)，\(MF\parallel DB\)。

## q2s2：第（II）②，建立 \(PB-PM\) 的最大值模型

- 显示对称轴 \(x=\frac m2\)，点 \(P(\frac m2,y)\)，以及线段 \(PB,PM\)。
- 使用局部控制变量 \(t\) 表示 \(P\) 在对称轴上的高度比例 \(y=tm\)，让学生拖动观察 \(PB-PM\) 的变化。

## q2s3：第（II）②，最大值状态与最终答案

- 锁定 \(t=\frac32\)，即 \(P(\frac m2,\frac32m)\)。
- 由最大值 \(m\sqrt2=\frac{3\sqrt2}{4}\) 锁定 \(m=\frac34\)。
- 标出最终抛物线 \(y=-2x^2+\frac32x+\frac94\) 和 \(P(\frac38,\frac98)\)。

# JSON 对齐说明

- `geometry-spec.expressionEnv` 先定义第（I）问系数 `a1,b1,c1`，再定义第（II）问参数族 `a,b,c,n,axisX,pY`。
- 第（I）问曲线使用 `parabolaPart1`，第（II）问曲线使用 `parabolaMain`。
- 第②问的 `P` 默认放在最大值状态 \(P(\frac m2,\frac32m)\)；在 `q2s2` 中通过局部控制临时覆盖为 \(P(\frac m2,tm)\)。
- `lesson-data.steps`、`policies`、`stepLabels` 与 `step-decorations.steps` 使用同一组 id：`i1s1,q1s1,q1s2,q2s1,q2s2,q2s3`。
